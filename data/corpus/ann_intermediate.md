# Artificial Neural Networks (ANN)

## Backpropagation and Gradient Descent
Backpropagation is the primary algorithm used for training feedforward neural architectures. It operates in two distinct phases: a forward pass and a backward pass. During the forward pass, input data flows through the network layers to produce a prediction, which is then compared against the actual target value using a loss function. The backward pass calculates the gradient of this loss function with respect to every weight and bias in the network by applying the calculus chain rule from the output layer back to the input layer. These calculated gradients dictate exactly how each parameter should be adjusted to minimize the error. Without this reverse propagation of error signals, gradient descent optimization would be impossible for deep multi-layer architectures, as the network would have no mathematical mechanism to assign blame to hidden neurons for their contribution to the final output error.

## Activation Functions and Non-Linearity
Activation functions are mathematical operations applied to the weighted sum of inputs at each neuron before passing the signal to the next layer. Their fundamental purpose is to introduce non-linearity into the network's mapping capabilities. If a network relied solely on linear operations, multiple layers would mathematically collapse into a single linear transformation, rendering the architecture incapable of solving complex, non-linear problems like the XOR logic gate or image classification. Common choices include the Rectified Linear Unit (ReLU), which outputs the input directly if positive and zero otherwise, and the Sigmoid function, which squashes values between zero and one. While Sigmoid was historically popular, it frequently suffers from vanishing gradients in deep architectures. ReLU mitigates this by allowing unattenuated gradient flow for positive values, making it the default choice for hidden layers in modern architectures.

## The Multi-Layer Perceptron (MLP)
The original perceptron algorithm consists of a single layer of input nodes connected directly to an output node. While historically significant, a single-layer perceptron is fundamentally restricted to learning linearly separable patterns; it fails completely when data points cannot be separated by a single straight line or hyperplane. The Multi-Layer Perceptron (MLP) overcomes this geometric limitation by introducing one or more hidden layers between the inputs and outputs. Each neuron in a hidden layer acts as a feature extractor, combining raw inputs into higher-level representations. This hierarchical stacking of dense, fully connected layers allows the MLP to approximate any continuous mathematical function, a principle known as the Universal Approximation Theorem. However, because every neuron connects to every neuron in the subsequent layer, MLPs scale poorly to high-dimensional data like images, leading to parameter explosion and severe overfitting.
