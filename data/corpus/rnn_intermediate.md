# Recurrent Neural Networks (RNN)

## The Hidden State and Sequential Memory
Recurrent architectures are explicitly designed to process sequential data by maintaining an internal memory mechanism called the hidden state. Unlike standard feedforward models that process fixed-size inputs in complete isolation, these networks contain cyclic connections. At each time step, the network receives both the current sequence element and the hidden state vector from the immediately preceding time step. This recurrent loop allows information to persist across the sequence, ensuring that the network's output at time step T is heavily influenced by the context of all inputs processed from time step 0 to T-1. This architectural design makes them naturally suited for tasks involving temporal dependencies, such as natural language translation, speech recognition, and stock price forecasting, where the meaning of a current input depends entirely on what came before it.

## Backpropagation Through Time (BPTT)
Backpropagation Through Time is the specialized optimization algorithm required to train networks with cyclic dependencies. To calculate gradients, the recurrent architecture must first be conceptually unrolled across the entire input sequence, transforming it into a deep feedforward network where each layer represents a specific time step. During the forward pass, inputs are processed sequentially to accumulate the final loss. During the backward pass, the error gradient is propagated backwards from the final time step all the way to the first. Because the exact same set of weights is reused at every single time step, the gradients calculated for each step are summed together to update the shared parameters. This unrolling process is highly memory-intensive for long sequences and mathematically exposes the architecture to severe gradient scaling issues during the reverse accumulation phase.

## The Vanishing Gradient Problem
The vanishing gradient problem is a catastrophic mathematical failure mode that prevents standard recurrent architectures from learning long-term dependencies. When the error signal is propagated backward through an unrolled network over dozens or hundreds of time steps, it repeatedly passes through the same weight matrices and non-linear activation functions like Tanh. If the singular values of the recurrent weight matrix are less than one, the continuous chain-rule multiplications cause the gradient to shrink exponentially toward zero. By the time the error signal reaches the earliest time steps in the sequence, the gradient is essentially zero, meaning the network weights cannot be updated to connect an early input to a late output. This physical limitation means standard recurrent models suffer from short-term memory, completely forgetting critical context from the beginning of a long paragraph or audio sequence.
